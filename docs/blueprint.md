# **App Name**: Streamlined HR Assistant

## Core Features:

- Easy Leave Submission: Simplified interface for leave request submission.
- Attendance View: Quick access to attendance records for each employee.
- HR Dashboard: Centralized dashboard for accessing key HR functions.
- Job Board: Job Board displaying active job openings.
- AI Career Suggestion Tool: Suggest personalized career development plans for employees based on their roles and skills using an AI tool.

## Style Guidelines:

- Primary color: Light navy (#465975) to create a sense of calm, focus, and reliability.
- Background color: Off-white (#F2F3F4) for a clean and professional feel.
- Accent color: Slate blue (#758AA2) to provide contrast without disrupting the serious mood.
- Font pairing: 'Space Grotesk' sans-serif for headers, and 'Inter' sans-serif for body text.
- Clean and professional icons that complement the overall minimalistic design.
- A grid-based layout that is responsive and adapts to different screen sizes.
- Subtle animations on button clicks and form submissions.
